=== Foreword

Contexts and Dependency Injection 2.0 (link:https://jcp.org/en/jsr/detail?id=365[JSR-365^]) is an update to Contexts and Dependency Injection 1.2 (link:https://jcp.org/en/jsr/detail?id=346[JSR-346^]).

A full changelog will be found link:https://issues.jboss.org/secure/ReleaseNote.jspa?version=12323655&styleName=Html&projectId=12311062[here^].

Starting with version 2.0 CDI targets Java SE and Java EE platforms.
CDI in a Java EE container is mostly a superset of CDI in Java SE, except for features explicitly stating otherwise.

=== Organisation of this document

This document is organized in 3 parts:

* An introduction (this part), whose not part of the specification but introduce CDI concepts and give examples.
* The CDI specification for Java SE: <<part_1>>.
* The CDI specification for Java EE: <<part_2>>, which extends the first part.
